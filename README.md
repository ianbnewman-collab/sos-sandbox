# 2nd Copythorne (Stanley's Own) Scout Group — Website Prototype

Static HTML prototype for Warren Turvey's sign-off review.

**Live preview:** https://ianbnewman-collab.github.io/homepage.html

## How to navigate

Start at `homepage.html`. The **Sections** menu in the nav bar drops down to all section hub pages. From the Scouts hub you can reach the King Troop and Everett Troop pages directly, or via the Sections dropdown on any page.

## Technical notes

- Files are Claude Design standalone HTML exports — serve as-is, no build step needed
- `.nojekyll` disables GitHub Pages' Jekyll processing so files are served exactly as uploaded
- See `WIRING_LOG.md` for the full audit trail of changes made during wiring
- See `patch_prototype.py` if re-patching is ever needed
